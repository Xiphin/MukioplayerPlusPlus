编译环境:
Flex SDK 4.0
运行需要FlashPlayer 10.0.0

Ant编译方法:
请安装ant，进入cmd模式，cd 项目目录， ant
使用ant编译前,把build.properties.sample改名为build.properties
并修改其中的Flex SDK目录设置

FlashBuilder编译方法:
创建一个新的Flex项目
指定路径为源代码路径,Flex SDK选4.0或更新的版本
编译前指定程序入口 src/MukioPlayerPlus.mxml
